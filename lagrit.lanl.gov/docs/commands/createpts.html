CREATEPTS

This command adds points to the mesh. For some special cases, it will also produce element connectivity.

Createpts is a wrapper for all the 'rz' type commands found elsewhere in the documentation.

createpts/xyz|rtz|rtp|line
createpts/brick
createpts/interp
createpts/sphere
createpts/random
createpts/vector
createpts/voronoi
createpts/median
createpts/amr
